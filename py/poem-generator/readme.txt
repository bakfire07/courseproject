Poem generator is based on code by schollz.

You need to download the repository from :https://github.com/schollz/poetry-generator/archive/master.zip

After downloading, please follow the instructions from: https://github.com/schollz/poetry-generator

Once the setup is done, please copy Poem Generator.ipynb to poetry-generator folder and run the jupyter notebook command. The details for installing jupyter are given under chat folder.
